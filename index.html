TUDO QUE VOCÊ PRECISA

Cada funcionalidade foi pensada para a melhor experiência gamer.

Biblioteca Inteligente

Detecção automática de jogos instalados. Metadados, banners e capas baixados automaticamente via RAWG e SteamGridDB. Interface estilo PS4 com carrossel animado.

Rastreamento de Horas

Detecta automaticamente quando você inicia e fecha um jogo. Contabiliza horas, minutos e sessões individuais. Histórico completo de cada partida.

Destaque

Sync Automático de Saves

Monitora sua pasta de saves em tempo real. Quando qualquer arquivo é alterado, faz commit e push automático para seu repositório GitHub privado. Nunca mais perca um save.

Backup Seguro

Seus saves ficam versionados no Git. Pode reverter para qualquer ponto no histórico. Conflitos detectados automaticamente. Criptografa seu token localmente.

Busca e Categorias

Encontre qualquer jogo instantaneamente. Organize por categorias, favoritos, gêneros e plataforma. Ordenação inteligente por recentemente jogado.

Totalmente Configurável

Tema escuro moderno. Configurações granulares. Logs detalhados. Notificações toast em tempo real. Interface responsiva em tela cheia.

Simples

CÓDIGO LIMPO

Modular, escalável e profissional. Separação total entre UI e lógica de negócio.

Estrutura de Módulos
nexus_launcher/
├─ core/ ← settings, constants, lifecycle
├─ database/ ← engine, models, migrations
├─ models/ ← dataclasses, DTOs
├─ services/ ← metadata, images, tracker
├─ game_detection/ ← auto-detect installed games
├─ save_monitor/ ← watchdog real-time
├─ github_sync/ ← git auto-commit & push
├─ ui/ ← PySide6, QSS, widgets
├─ utils/ ← logger, cache, helpers
└─ main.py
Signals & Slots
Comunicação entre UI e lógica via Qt signals. Zero acoplamento.
Threading Assíncrono
Play tracker, file watcher e sync rodam em threads separadas. UI nunca congela.
Debounce Inteligente
Múltiplas alterações de arquivo geram apenas 1 commit. Sem spam no Git.
Cache em Disco
Imagens e metadados cacheados localmente. Uma única requisição por jogo.
SQLAlchemy ORM
Banco relacional com migrations, session management e WAL mode.
